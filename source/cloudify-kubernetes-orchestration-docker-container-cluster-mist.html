Easy Kubernetes Orchestration Service With Cloudify on Mist.io

About The Service

Cloudify and Mist.io are proud to offer Kubernetes Orchestration-as-a-Service for quick, hassle free provisioning and configuring of a Kubernetes cluster. The orchestration service simplifies the provisioning of complex infrastructure and applications across heterogeneous environments.

We make setting up a Kubernetes cluster on AWS or OpenStack simple. With just a single click, a cluster can be set up in minutes, making it easy for users to get started with Kubernetes. New providers will be supported in the future.

How It Works

The cloud orchestration service, based on Cloudify, the open-source, TOSCA-based orchestration platform enables you to create a single, reusable blueprint to provision the same application across multiple cloud providers, data centers, or hybrid environments.

Rule based auto-scaling tools make it easy to scale up and down the cluster when necessary.

The service combines cloud orchestration with Mist.io features such as monitoring, scheduling, RBAC, and/or APIs, to create automated and dynamic workflows and policies that respond to system events in real time.
