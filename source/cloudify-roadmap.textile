h1. *Cloudify Roadmap | Version 3.3*

h2. Target Date: Q4 2015

h1. <i class="icon-star"></i> *Features*

h2. <i class="icon-star-empty"></i> *Manager and Agents Offline Installation*

Cloudify Manager and agents are shipped and can be installed from a local repository without internet connection.


h2. <i class="icon-star-empty"></i> *Distributed Manager Blueprint*

Allows separating deployment of Cloudify Manager components on multiple nodes.

h2. <i class="icon-star-empty"></i> *Cloudify Manager Images*

Deploy the manager using an Image.

h2. <i class="icon-star-empty"></i> *Offline plugin installation*

h2. <i class="icon-star-empty"></i> *Agent installation via cloud-init allowing SSH-less agent installation*

h2. <i class="icon-star-empty"></i> *Improved manager process management for more scalability*
