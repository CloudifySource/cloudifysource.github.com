Cloudify Guided Tour
11 December 2015

If you're a developer in the tri-state area looking to learn more about cloud automation and orchestration, join us! We'd like to invite you to a one-day technical deep-dive on technology agnostic cloud orchestration. (<~ Buzzword abuse!)
We'll start with the fundamentals - from the buzzwords through what's involved with deploying and orchestrating real apps on the cloud, down to the fun parts like security groups and networking.

In this workshop you'll learn how to model your application topology once in simple YAML with Cloudify, and deploy it to any cloud, all while maintaining your existing tool chain.
